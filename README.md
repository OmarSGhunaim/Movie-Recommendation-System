# Movie Recommendation System

## Objective
Recommend movies based on genre similarity.

## What this project does
- Takes a movie as input
- Finds its genre
- Suggests similar movies

## Steps performed
1. Loaded movie dataset
2. Created recommendation function
3. Filtered movies by genre
4. Returned similar movies

## Example
Input: Avengers  
Output: Iron Man, Batman, Superman

## Tools used
- Python
- Pandas
